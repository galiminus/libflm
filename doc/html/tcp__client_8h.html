Create a new TCPClient object and connect it to host:port.

Note that the connection will be handled in the background, thus a connection error is more likely to appear in the error handler.

Parameters:
monitorA pointer to the Monitor object used to monitor socket availability.
cn_handlerHandler called on connect.
rd_handlerHandler called on read.
wr_handlerHandler called on write.
cl_handlerHandler called on close.
er_handlerHandler called on error.
dataUser defined data.
hostThe hostname to connect to.
portThe port to connect to.

A new TCPClient object.

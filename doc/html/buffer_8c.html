flm_Buffer* flm_BufferPrintf ( const char *  format,
  ... 
)

Create a new buffer object and fill it with a formatted string.

This function is the equivalent of printf() for buffers. Internally, it uses the vsnprintf() function so all the formatting rules are the same.

You should never use an user input as format string, see the BUGS section of the printf() manual page.

Parameters:
formatA printf()-style format string
Returns:
A pointer to a new buffer object
